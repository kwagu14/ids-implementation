Release Notes for

STM32L552E-EV Board Drivers

Copyright © 2019 STMicroelectronics

Purpose

This driver provides a set of functions to manage:

  • LEDs, push-button and joystick
  • Audio
  • TFT-LCD module with touchscreen
  • SD card
  • OctoSPI NOR Flash and HyperRAM memories
  • SRAM memory
  • Power measurement
  • USB Power Delivery
